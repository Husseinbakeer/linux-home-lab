# Linux Admin Scripts

Bash scripts for everyday Linux server tasks.  
Created while preparing for junior Linux Administrator and DevOps roles in Cairo, Egypt.

## What's inside

- `backup.sh`  
  Creates compressed backups of `/etc` and `/home`.

- `sysinfo.sh`  
  Quick overview: hostname, uptime, CPU, memory, disk usage, top processes.

- `logwatch.sh`  
  Shows recent errors and failed login attempts from syslog.

## How to use

```bash

# Run
./backup.sh
